# AllStars - Fix Buckets - ESX

- Cette version corrige les instances fantomes sans casser les interieurs.

#Fonctions :

- detection joueur bloque dans bucket

- verification interieur / exterieur

- reset intelligent

- fix au spawn

- fix reconnect

- commande admin

- commande admin sur joueur

- logs debug

- protection interieurs

( teste sur es_extended 1.13.4 )


#Installation

-- Créer dossier :

resources/[utils]/allstars_bucketsfix

-- Ajouter dans server.cfg

ensure qs_bucketfix

-- Redemarrer serveur.
